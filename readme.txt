**English**
-----------


1. Upload ALL the folder `adsense-google-widget` to folder `/wp-content/plugins/`.


2. Activate the plugin Lys Creation Google Adsense (Plugin) Widget through the 'Plugins' menu in the administration page of your WordPress.


3. Configure the Google Adsense Widget 'Settings'.

== Frequently Asked Questions ==

**English**
-----------

= Can I use this plugin to only display ads from Google Adsense? =

Sure, you can use this plugin to display Google ads Andsense. Remember with **Lys Creation in Google Adsense (Plugin) widget** you always keep control.

= What is Google Adsense? =

Google AdSense is a free program that allows online publishers to earn revenue by displaying relevant ads in a wide variety of online content.

= How do I display ads from Google Adsense? =

Just you need to be an affiliate of [Google Adsense](http://www.google.com/adsense), **it's free** and install this plugin.

= The plugin does not work! =

It can take few minutes (5 to 10min) from Google to be done on your blog. The plugin **requires PHP 5** or higher to function. If your blog is using version 4, the plugin does not display the ads.

= I got this error: Warning: array_merge() [function.array-merge]: Argument #2 is not an array in ...../wp-includes/widgets.php on line 777 =

Unfortunately your Theme got an error/issue, try to contact the developper to resolve it. Now you have to desactivate this plugin, you can NOT use it on this theme.
